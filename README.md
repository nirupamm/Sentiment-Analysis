# Sentiment Analysis Project

A machine learning web application for movie review sentiment classification using three different algorithms.

## Features
- Logistic Regression, Naive Bayes, and Decision Tree models
- Streamlit web interface for real-time sentiment analysis
- Multi-model comparison capabilities

## Installation
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Run training script: `python train_models.py`
4. Launch app: `streamlit run sentiment_app.py`

## Technologies
- Python 3.x
- scikit-learn
- Streamlit
- pandas
- joblib
